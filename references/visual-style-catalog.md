# Visual Style Catalog

Use this reference to guide poster style selection. These options are synthesized from current web-design inspiration patterns: award-site hero pages, portfolio grids, interaction-heavy landing pages, editorial websites, and curated design galleries. The bundled cards are original HTML/CSS poster references, not copied website screenshots.

Show the overview image when possible:

- `assets/style-reference-cards/style-reference-overview.png`

Individual reference cards live in `assets/style-reference-cards/`.

The reference cards are poster-style visual references, not PPT templates. Use them to choose a poster direction, then build the final poster as HTML/CSS with real text.

Users may either choose one of these styles, mix two styles, or describe the desired visual style in their own words. Treat a custom style description as a design brief: extract mood, layout density, typography, color, texture, and imagery cues, then map it to the closest built-in style or create a custom Zongteng-compliant HTML/CSS direction.

## Style Options

### 01 极简商务 / Minimal Business

Reference: `assets/style-reference-cards/01-minimal-business.png`

Use for formal notices, leadership messages, strategy, external-facing brand statements, and infrastructure/service positioning. Borrow from premium SaaS/editorial hero pages: large whitespace, strong hierarchy, restrained blue/orange accents, and few words.

### 02 新粗野主义 / Neo Brutalism

Reference: `assets/style-reference-cards/02-neo-brutalist.png`

Use for young, energetic internal campaigns that need impact. Borrow from brutalist web pages and poster-like landing pages: hard borders, visible blocks, high contrast, sticker-like labels, and direct typography. Keep the Zongteng logo clean and untouched so the style does not damage brand integrity.

### 03 新野兽派色彩 / Neo Fauvist Color

Reference: `assets/style-reference-cards/03-neo-fauvist.png`

Use for festivals, activities, culture campaigns, and lightweight internal posters. Borrow from color-forward campaign websites: saturated organic color fields and emotional contrast, anchored with Zongteng Blue/Orange or value IP colors.

### 04 瑞士网格 / Swiss Grid

Reference: `assets/style-reference-cards/04-swiss-grid.png`

Use for reports, process notices, policy communication, meeting screens, and information-heavy posters. Borrow from typographic portfolio and museum/editorial grids: visible alignment, strict columns, numbered sections, and clean typography.

### 05 Bento 信息卡 / Bento Grid

Reference: `assets/style-reference-cards/05-bento-data.png`

Use for multi-point announcements, benefits, data, agendas, service introductions, and long posters. Borrow from product landing pages and dashboard sections: stable cards, clear metrics, compact modules, and controlled spacing.

### 06 软萌 3D IP / Soft 3D Mascot

Reference: `assets/style-reference-cards/06-soft-3d-ip.png`

Use for values, employee engagement, recognition, internal culture, and friendly HR/culture posters. Borrow from character-led product websites: soft depth, rounded forms, approachable hero composition, and clean value chips. Match values to mascots from `values-ip.md`.

### 07 液态玻璃科技 / Liquid Glass Tech

Reference: `assets/style-reference-cards/07-glass-tech.png`

Use for digitalization, systems, AI, HRAS platforms, technology service, and dashboard-like communications. Borrow from modern AI/SaaS landing pages: translucent panels, deep blue gradients, luminous data layers, and high contrast.

### 08 杂志拼贴 / Editorial Collage

Reference: `assets/style-reference-cards/08-editorial-collage.png`

Use for employee stories, team profiles, event recaps, campaign storytelling, and culture features. Borrow from editorial/portfolio websites: layered image blocks, paper/tape details, headline typography, and brand color accents.

### 09 赛博数据 / Neon Data Dash

Reference: `assets/style-reference-cards/09-cyber-neon.png`

Use for tech launches, battle reports, speed/efficiency themes, AI/data topics, or high-energy event countdowns. Borrow from dark-mode data products and motion-heavy tech sites: dark background, electric lines, grid, neon accents, and orange highlights.

### 10 触感纸艺 / Tactile Paper

Reference: `assets/style-reference-cards/10-tactile-paper.png`

Use for care, wellbeing, employee letters, holiday greetings, culture warmth, and reflective topics. Borrow from tactile brand sites and handmade editorial layouts: paper texture, soft shadows, human warmth, and modest brand marks.

### 11 液态超现实 / Liquid Gradient World

Reference: `assets/style-reference-cards/11-surreal-gradient.png`

Use for future vision, innovation, festival visuals, campaign teasers, and abstract brand atmosphere. Borrow from immersive WebGL/gradient hero pages: fluid forms, atmospheric color, depth, and crisp real text on top.

### 12 温暖人文 / Warm Humanist

Reference: `assets/style-reference-cards/12-warm-humanist.png`

Use for employee stories, onboarding, culture care, recognition, and real workplace moments. Borrow from warm people-first product/portfolio pages: photo-led structure, soft panels, gentle rhythm, and simple typography.

## Custom Style Handling

If the user describes a desired style instead of choosing a card:

- Map concrete descriptors to the closest catalog style. Example: "像高端科技官网首屏" maps to Liquid Glass Tech + Minimal Business; "像设计工作室作品集" maps to Editorial Collage + Swiss Grid.
- Preserve non-negotiable brand rules: exact logo assets, Zongteng Blue/Orange, readable Chinese text, value/IP mapping, and HTML/CSS output.
- Use external websites only as inspiration for mood and composition. Do not copy screenshots, proprietary illustrations, or exact layouts into generated posters.
- If the user names a specific website or brand style, describe the adaptation as "inspired by" and rebuild it as an original Zongteng poster.

## Selection Rules

- For formal audiences, default to Minimal Business, Swiss Grid, Bento Grid, or Corporate Blue Orange.
- For all-employee culture communication, default to Soft 3D Mascot, Warm Humanist, Tactile Paper, or Event Campaign.
- For youth-oriented internal campaigns, use Neo Brutalism, Neo Fauvist Color, Liquid Gradient World, or Neon Data Dash.
- For HRAS or digital service posters, use Liquid Glass Tech, Bento Grid, Minimal Business, or Warm Humanist.
- For long posters, prefer Bento Grid, Swiss Grid, Editorial Collage, or Minimal Business.
- If the user asks for "高级", "稳重", or "商务", avoid Neo Brutalism, Neon Data Dash, and overly saturated Neo Fauvist Color unless explicitly requested.
- If the user asks for "活泼", "年轻", or "有冲击", avoid purely minimal corporate layouts.
