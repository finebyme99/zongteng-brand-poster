# Visual Style Catalog

Use this reference to guide poster style selection. These options are based on current design directions such as imperfect/tactile design, neo-brutalism, high-saturation color, Bento layouts, expressive typography, 3D character branding, glassy tech layers, collage/storytelling, and minimalist fatigue responses.

Show the overview image when possible:

- `assets/style-reference-cards/style-reference-overview.png`

Individual reference cards live in `assets/style-reference-cards/`.

## Style Options

### 01 极简商务 / Minimal Business

Reference: `assets/style-reference-cards/01-minimal-business.png`

Use for formal notices, leadership messages, strategy, external-facing brand statements, and infrastructure/service positioning. Keep large whitespace, strong hierarchy, restrained blue/orange accents, and few words.

### 02 新粗野主义 / Neo Brutalism

Reference: `assets/style-reference-cards/02-neo-brutalist.png`

Use for young, energetic internal campaigns that need impact. Use hard borders, visible blocks, high contrast, and direct typography. Keep the Zongteng logo clean and untouched so the style does not damage brand integrity.

### 03 新野兽派色彩 / Neo Fauvist Color

Reference: `assets/style-reference-cards/03-neo-fauvist.png`

Use for festivals, activities, culture campaigns, and lightweight internal posters. Use saturated color fields and emotional contrast, but anchor the system with Zongteng Blue/Orange or value IP colors.

### 04 瑞士网格 / Swiss Grid

Reference: `assets/style-reference-cards/04-swiss-grid.png`

Use for reports, process notices, policy communication, meeting screens, and information-heavy posters. Use visible alignment, strict columns, numbered sections, and clean typography.

### 05 Bento 信息卡 / Bento Grid

Reference: `assets/style-reference-cards/05-bento-data.png`

Use for multi-point announcements, benefits, data, agendas, service introductions, and long posters. Split content into stable cards and avoid overcrowding.

### 06 软萌 3D IP / Soft 3D Mascot

Reference: `assets/style-reference-cards/06-soft-3d-ip.png`

Use for values, employee engagement, recognition, internal culture, and friendly HR/culture posters. Match values to mascots from `values-ip.md`.

### 07 玻璃科技感 / Glass Tech

Reference: `assets/style-reference-cards/07-glass-tech.png`

Use for digitalization, systems, AI, HRAS platforms, technology service, and dashboard-like communications. Use translucent panels over deep blue gradients. Keep contrast high.

### 08 杂志拼贴 / Editorial Collage

Reference: `assets/style-reference-cards/08-editorial-collage.png`

Use for employee stories, team profiles, event recaps, campaign storytelling, and culture features. Combine photo blocks, torn-paper-like panels, headline typography, and brand color accents.

### 09 赛博霓虹 / Cyber Neon

Reference: `assets/style-reference-cards/09-cyber-neon.png`

Use for tech launches, battle reports, speed/efficiency themes, AI/data topics, or high-energy event countdowns. Use dark background, electric lines, grid, neon accents, and orange highlights.

### 10 触感纸艺 / Tactile Paper

Reference: `assets/style-reference-cards/10-tactile-paper.png`

Use for care, wellbeing, employee letters, holiday greetings, culture warmth, and reflective topics. Use paper texture, human warmth, and modest brand marks.

### 11 超现实渐变 / Surreal Gradient

Reference: `assets/style-reference-cards/11-surreal-gradient.png`

Use for future vision, innovation, festival visuals, campaign teasers, and abstract brand atmosphere. Keep text crisp on top of gradients.

### 12 温暖人文 / Warm Humanist

Reference: `assets/style-reference-cards/12-warm-humanist.png`

Use for employee stories, onboarding, culture care, recognition, and real workplace moments. Pair soft photo-like shapes or portraits with simple typography.

## Selection Rules

- For formal audiences, default to Minimal Business, Swiss Grid, Bento Grid, or Corporate Blue Orange.
- For all-employee culture communication, default to Soft 3D Mascot, Warm Humanist, Tactile Paper, or Event Campaign.
- For youth-oriented internal campaigns, use Neo Brutalism, Neo Fauvist Color, Surreal Gradient, or Cyber Neon.
- For HRAS or digital service posters, use Glass Tech, Bento Grid, Minimal Business, or Warm Humanist.
- For long posters, prefer Bento Grid, Swiss Grid, Editorial Collage, or Minimal Business.
- If the user asks for "高级", "稳重", or "商务", avoid Neo Brutalism, Cyber Neon, and overly saturated Neo Fauvist Color unless explicitly requested.
- If the user asks for "活泼", "年轻", or "有冲击", avoid purely minimal corporate layouts.
