# Design Quality Bar

Use this reference before creating any final poster. The goal is a designed poster, not a slide, dashboard, form, or plain information card.

## Non-Negotiable Standard

Every poster must have a clear visual concept and a dominant first-read composition. If the result looks like a PPT cover, an internal notice template, a dashboard, or a grid of equal cards, discard it and redesign.

## Required Poster Qualities

- Use a hero visual system: oversized typography, strong campaign graphic, mascot/IP scene, photo/collage, brush stroke, texture field, dimensional object, or cinematic composition.
- Create depth and rhythm: overlapping layers, foreground/background separation, diagonals, curves, texture, lighting, cutouts, or motion traces.
- Keep a strong first viewport: the title and theme should be readable immediately, but the layout should also feel like a designed campaign visual.
- Use information blocks as supporting details only. They should not be the main visual idea.
- Use Zongteng Blue and Orange with intent: blue as structure/energy field, orange as impact/CTA/motion highlight.
- Use exact logo and IP assets from `assets/`; never redraw brand marks.
- Keep Chinese text as real HTML text. If expressive calligraphy or brush title effects are needed, emulate them with CSS/text layering or use them as non-critical decorative background, while keeping the official title in readable HTML text.

## Anti-PPT Rules

Do not produce:

- A centered title with several equal rectangular cards below it.
- A pale grid background with generic circles, rings, and small dots as the main visual.
- A dashboard/report layout unless the user explicitly asks for a report poster.
- A slide-like 16:9 frame that only arranges text boxes and icons.
- A layout where every element is aligned in neat blocks with no visual tension, hero image, texture, or campaign energy.
- A poster whose main visual could be mistaken for a PowerPoint agenda slide.

## Campaign Poster Patterns

Use these patterns when the topic is activity, competition, recognition, values, or culture:

- **Brush Energy**: large expressive title, blue/orange brush strokes, angled motion, mascot/IP or object cutouts, bottom information strip.
- **Mascot Campaign**: values/IP characters as the hero, big slogan, energetic background, compact CTA and schedule section.
- **Editorial Impact**: oversized headline, cutout photo/illustration, layered paper or image blocks, strong brand-color accents.
- **Tech Arena**: dark or deep-blue field, luminous data/motion lines, 3D/AI object, orange CTA burst, structured detail band.
- **Bento Support Layer**: use Bento cards only as secondary details after a strong hero visual; never make Bento cards the whole poster.

## Self-Check Before Export

Answer yes to all before delivering:

1. Does it look like a designed poster rather than a PPT slide?
2. Is there a dominant hero visual or campaign composition?
3. Are Zongteng brand elements visible within the first read?
4. Are supporting details secondary to the main visual?
5. Is all important Chinese text real, readable HTML text?
6. Would this feel acceptable next to a professionally designed internal campaign poster?
