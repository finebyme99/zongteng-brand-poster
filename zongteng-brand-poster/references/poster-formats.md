# Poster Formats

Use this reference when choosing layout size, ratio, and format behavior.

All formats below are poster canvases. Do not interpret `16:9` as a presentation slide deck; it is a single screen poster.

## Default Poster

When the user does not specify a size, use a vertical internal/social poster:

- Ratio: 3:4.
- Suggested size: `1080 x 1440`.
- Use one main headline, one supporting line, date/location if needed, and logo.
- Keep enough whitespace for mobile and Feishu preview readability.

## Long Poster

Use for detailed announcements, process guides, event agendas, campaign explainers, or multi-section culture content.

Suggested sizes:

- `1080 x 1920` for short long-form mobile poster.
- `1080 x 2400` or taller for detailed content.

Rules:

- Build a top hero section, 2-5 content sections, and a closing/logo footer.
- Repeat small brand anchors between sections.
- Use Zongteng Blue as section anchor and Orange for emphasis.
- Keep paragraphs short. Convert details into numbered blocks, timeline strips, badges, or cards.
- Do not simply scale a 16:9 layout vertically.

## Meeting-Room Projection

Use for Feishu/meeting-room screen posters.

Target:

- Ratio: 16:9.
- Suggested size: `1920 x 1080`.

Rules:

- Reserve the top area for meeting metadata such as projection code, calendar/organizer/time, date/time, and logo.
- Keep the main message in the visual core area.
- Keep text and critical images away from edges.
- Use dark or muted backgrounds when possible.
- Use large type readable from a distance.
- Avoid dense paragraphs.

## Square / Social

Use for Feishu posts, internal feed cards, and thumbnails:

- Ratio: 1:1.
- Suggested size: `1080 x 1080`.
- Use a strong title and one visual focus.
- Keep the logo small but visible.

## Horizontal Banner

Use for intranet, Feishu group header, event banner, or screen top image:

- Ratio: 16:9, 2:1, or 3:1 depending on platform.
- Keep the title left or center, logo in a stable corner, and brand pattern as background.

## Print

Use A4/A3 only when the user asks for print:

- A4 portrait: internal notices and handouts.
- A3 portrait: public area posters.
- Ensure logo size follows VI minimum size guidance.
- Use high contrast and avoid tiny text.

## Output Source

Build these formats as HTML/CSS poster canvases and export to PNG/PDF. Keep Chinese text as real DOM text until export.
