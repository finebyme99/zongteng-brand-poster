# Values IP

Use this reference for Zongteng culture, values, and mascot posters.

## Value Mapping

- Respect / 尊重: Panda.
- Excellence / 卓越: Eagle.
- Pragmatism / 务实: Elephant.
- Win-Win / 共赢: Bee.

Use the matching mascot whenever a poster focuses on one value. Do not swap mascots across values.

## Visual Language

The value IP system is friendly, rounded, bright, and employee-facing:

- Rounded cards.
- Large English value words in the background.
- Soft shadows.
- Bright value-specific backgrounds.
- 3D mascot scenes or mascot guide images.
- Footer labels such as `ZONGTENG GROUP` and `VALUE IP IMAGE` are acceptable for culture cards.

Use this style for:

- Value posters.
- Culture campaigns.
- Internal recognition.
- Employee activities.
- Festival/internal greetings.
- People and team engagement.

Avoid this style for:

- Board, legal, compliance, formal investor, or serious external corporate communication.
- High-authority strategy posters unless the user asks for a warm culture treatment.
